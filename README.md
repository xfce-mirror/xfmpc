# xfmpc

xfmpc is a graphical GTK+ Music Player Daemon (MPD) client focusing on low footprint.

### Installation

From source code repository: 

    % cd xfmpc
    % ./autogen.sh
    % make
    % make install

From release tarball:

    % tar xf xfmpc-<version>.tar.bz2
    % cd xfmpc-<version>
    % ./configure
    % make
    % make install
